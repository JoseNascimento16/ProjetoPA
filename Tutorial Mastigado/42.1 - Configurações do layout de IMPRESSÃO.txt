- No media.css incluimos:

@media print {

    body {
        background: #CCCCCC;
        background: none;
        -ms-zoom: 1.665;
        transform: scale(0.97);
      }

    @page {
        size: auto;
        margin: 1mm;
    }
}


# transform: scale(0.97); - Altera a proporção de escala do layout de impressão. No ppróprio layout tem outra opção para ajustar lá dentro a escala também

## size: auto; - Faz com que apareça no layout de impressão a opção de formato "retrato" ou "paisagem"

### margin: 1mm; - Determina as margens padrão do layout de impressão. Dentro do layout também há uma opção de configurar as margens.
