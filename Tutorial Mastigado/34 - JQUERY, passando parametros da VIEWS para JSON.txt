Imagine o cenário:
- Queremos utilizar uma variável que temos no HTML ou na views.py, só que no JQUERY do javascript.

Usaremos uma lista[].

- Na views.py criamos a lista

	ordens_lista=[]

  Depois adicionamos os valores a esta lista. No nosso caso, fizemos um FOR para capturar valores numericos de um objeto iteravel
	
	for elemento in ordens_iteravel:
            ordens_lista.append(elemento.identificacao_numerica)

	como resultado temos: ordens_lista=[1,2,3] por exemplo

  Depois passamos esta lista para o contexto normalmente:

	plano_a_exibir = {
        	'chave_lista_ordens' : ordens_lista
    	}
    
   	return render(request, 'acao-visualizacao.html', plano_a_exibir)

  Pronto nossa lista já é acessivel no HTML.


- Agora no HTML, passamos esta lista para um JSON através deste comando:

	{{ chave_lista_ordens|json_script:"id-do-json" }}

	Ou seja, criamos um script com o id "id-do-json", e que carrega como valor o conteudo de "chave_lista_ordens", a nossa lista.

	O local no html onde iremos por este comando não importa muito, pois a lista ja vem pronta


- Agora no documento .js que será carregado na página, iremos resgatar esta variável da seguinte forma:

	var lista_de_ordens = JSON.parse(document.getElementById('id-do-json').textContent);

	Esta variável "lista_de_ordens" recebe uma lista, portanto, ela também é uma lista

	Pronto, já temos a nossa variável em forma de lista [1,2,3] acessível pelo jQuery!!


Extra: No nosso projeto usamos uma espécie de FOR para iterar entre esses valores numericos, e assim, colocá-los como parte do nome das classes das tags HTML de uma tabela, alterando-as de forma dinamica conforme passamos o mouse por cima dos elementos na página:

$(document).ready(function(){

    var lista_de_ordens = JSON.parse(document.getElementById('id-do-json').textContent);
    // var testando = ["1","3"]
    // console.log(valor);
    // console.log(valor[0]);
    $( lista_de_ordens ).each(function(index,element) {
        // console.log( element );

        $('.tds-trs-' + element).hover(function(){
        $('.tr-dinamicos' + element).addClass('background-tr-acoes');
        }),
        
        $('.tds-trs-' + element).mouseleave(function(){
        $('.tr-dinamicos' + element).removeClass('background-tr-acoes');
        })

      });

    
    
    
    })
