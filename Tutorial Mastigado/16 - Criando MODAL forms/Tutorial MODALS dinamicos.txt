Para criar MODALS DINAMICOS teremos a seguinte estrutura:

- 1 estrutura de MODAL partial com seu HTML base (fundo preto e branco com botão de fechar)
- 1 ou 2 estruturas de CORPO DE FORMULARIO HTML que será mostrado no MODAL.
- 1 BOTÃO/LINK HTML que irá ativar o MODAL
- 1 arquivo .css para o MODAL
- 1 arquivo .css para o FORMULARIO
- 1 arquivo .js para a animação com codigo jQuery.

(todos disponibilizados nesta pasta)

(A ordem dos próximos passos não importa)

1 - Posicionar o HTML do MODAL na pasta partials do nosso projeto (ver exemplo 1)

2 - Criar uma subpasta "formularios" dentro da pasta partials (ver exemplo 2)
	- Colocar o html de cada formulario nesta pasta

3 - Na página em que queremos ativar o MODAL inserimos: (ver exemplo 3)
	- O código python que chama o MODAL-partial (de preferencia no fim do body, dentro do BLOCK CONTENT)
		{% include 'partials/modal-partial.html' %}
	- O html do botão/link que irá ativar o jquery e setar as propriedades do modal para que ele apareça

4 - Dentro do HTML do modal-partial inserimos:
	- Os códigos pythons que chamam os formularios (ver exemplo 4)
		{% include 'partials/formularios/form-plano.html' %}
           	{% include 'partials/formularios/form-cadastro.html' %}

5 - Colocar o arquivo modal.js na pasta static do projeto responsavel pelos arquivos .js (ver exemplo 5)

6 - Incluir os scripts nas chamadas do head no base.html. (ver exemplo 6)
	<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
	<script src="{% static 'js/modal.js' %}"></script>

7 - Incluir as chamadas de scripts no fim da pagina base.html também
	<script src="https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/4.1.3/js/bootstrap.bundle.min.js"></script>
        <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>
---------------------------------------------------------------

A lógica de funcionamento: (ver exemplo 7)

	- Inicialmente o Modal e os formularios terão suas propriedades css "display = none". 
	- Ao clicar-mos no botão/link, que terá uma classe ou id associado, o arquivo modals.js irá identificar este clique e irá trocar o valor desta propriedade para flex ou block.
	- Ao clicarmos no X para fechar, teremos o comportamento oposto.
