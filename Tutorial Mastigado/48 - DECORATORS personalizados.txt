Um decorator, é uma função que recebe uma outra função como atributo.

Como exemplo vamos criar uma para verificar se o usuário está ou não logado logado e redirecioná-lo para a pagina correta.

- Criamos um arquivo decorators.py

- Em decorators.py:

def usuario_nao_autenticado(funcao):
    def funcao_de_checagem(request, *args, **kwargs):
        if request.user.is_authenticated:
            return redirect('dashboard')
        else:
            return funcao(request, *args, *kwargs)

    return funcao_de_checagem

- na função da views que quisermos:

@usuario_nao_autenticado
def funcao_qualquer(request):
    ...

######

Claro que este exemplo serve só para mostrar como criar um decorator personalizado, pois ´django já possui um decorator para esta finalidade: @login_required
