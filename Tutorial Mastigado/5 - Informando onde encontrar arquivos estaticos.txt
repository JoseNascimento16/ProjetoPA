- Criando o folder /static

    SetupPrincipal > New Folder > static



Em SetupPrincipal > settings.py

- Criando o path de raiz (linha 122)
    
    STATIC_ROOT = os.path.join(BASE_DIR, 'STATIC')



- Criando o path de diretorio (linha 124)
    
    STATICFILES_DIRS = [
            os.path.join(BASE_DIR, 'SetupPrincipal/static')
            ]


- Informando ao Django para coletar esses arquivos na pasta static

    python manage.py collectstatic


* SE ADICIONAR NOVOS ARQUIVOS ESTATICOS, TALVEZ TENHA QUE REAPLICAR ESTE CODIGO!
