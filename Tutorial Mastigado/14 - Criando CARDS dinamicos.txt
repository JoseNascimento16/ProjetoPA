Para criar um card dinâmico faremos o seguinte.

1 - Criar o html do card. (Exemplo no .txt 14-1 CARD html)

2 - na função de renderização no arquivo views.py do app que estamos trabalhando vamos criar um dicionário de valores, e esse valores serão mostrados no card. E ao colocar o contexto "dados" na render, podemos passar as informações contidas no dicionario "planos" como valores no nosso HTML

Ex:

def planos_de_acao(request):
    planos = {
        1:'2019',
        2:'2020',
        3:'2021',
        4:'2022'
    }

    dados = {
        'identificacao_dos_planos' : planos
    }
    
    return render(request, 'planos_de_acao.html', dados)

3 - Só precisamos agora embedar 1 unico card html em código python. Fazemos um "for" para varrer todas as chaves/valores do dicionario. E depois trocamos o valor estatico (ex: 2020) no html por {{valor}}

Ex:

{% for chave, valor in identificacao_dos_planos.items %}

            <div class="cards-planos">
                <p class="clear-2 itens-cards-planos">
                    <a href="#">
                        <i class="fas fa-edit margem-icones-menu"></i>{{ valor }}
                    </a>
                </p>
            </div>

{% endfor %}
