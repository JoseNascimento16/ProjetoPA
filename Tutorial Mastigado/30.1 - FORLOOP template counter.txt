- Caso queiramos capturar o estado de um loop FOR no template, temos o comando " forloop.counter0 "

Ex: Imagine que uma variavel de contexto do tipo querySet possui 10 objetos e queremos fazer alguma ação quando o FOR passar pelo 6º objeto:

Suponha que a variável tem o nome de 'chave_nomes', portanto ficariamos com essa estrutura:

{% for item in chave_nomes %}

   < faça algo...>

   {% if forloop.counter0 == 5 %}

   < faça algo mais...>
  
   {% endif %}

{% endfor %}

- Perceba que queriamos o 6º elemento, mas comparamos com o numero 5.
- Isso é porque o contador começa no 0, portanto, ao chegar no 5, este será o 6º elemento
