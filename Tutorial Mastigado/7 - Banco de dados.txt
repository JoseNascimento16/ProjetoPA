Para começar a trabalhar com banco de dados, vamos usar o Postgree

Depois de instala-lo, iremos executar o pgadmin 4 pelo menu iniciar.

Vamos inicia-lo clicar em Servers > PostgreeSQL > botao direito > criar > server
    Em nome: dbserver
    Em connection: localhost
    Em password: 123456
    Salvar.

Agora vamos instalar um modulo no pyhton para fazer essa conexão entre o Postgre e a nossa aplicação Django

No terminal já com o (venv) rodando vamos instalar o:
    pip install psycopg2
E também:
    pip install psycopg2-binary

Em SetupPrincipal > settings.py > DATABASES (linha78). Ficará da seguinte forma
    DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'SetupPrincipal',
        'USER': 'postgres',
        'PASSWORD': '123456'
        'HOST': 'localhost'
    }
}

Note que de todas essas informações, somente o NAME ainda não criamos lá no postgres
portanto vamos fazer isso.

Lá no Postgre no nosso dbserver botão direito vamos criar uma nova database de nome 'SetupPrincipal',

Pronto agora nosso banco de dados já esta pronto para começarmos a trabalhar nele.
