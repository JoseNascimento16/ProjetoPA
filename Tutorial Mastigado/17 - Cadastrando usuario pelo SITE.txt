Para este passo, é preferível que ja tenhamos pronto:

- Uma página ou MODAL contendo o formulário de cadastro

Como se trata de cadastro de usuarios, vamos utilizar o app de usuarios e teremos uma página de cadastro.html.
Queremos que no momento em que clicarmos no botão "Criar sua conta", a tabela "auth_user" deverá inserir esses dados e criar o novo usuário.

No formulário, devemos indicar o PATH, o metodo POST e iserir o TOKEN de segurança csrf logo abaixo:

	<form action="{% url 'cadastrar '%}" method="POST">
	{% csrf_token %}

em usuarios > urls.py devemos ter algo do tipo:
	
	path('cadastros/',views.cadastro, name='cadastrar'),

em usuarios > views.py
	
	a função responsável fica mais ou menos assim:

def cadastro(request):
    if request.method == 'POST':
        Nome_1 = request.POST['nome_1'] ## nome_1 é o valor do atributo name="nome_1" do input
        Nome_2 = request.POST['nome_2']
        Senha_1 = request.POST['password_1']
        Senha_2 = request.POST['password_2']

        if not Nome_1.strip() or not Nome_2.strip() :  ## validação
            print('O campo nome não pode ficar em branco')
            return redirect('cadastrar')

        if Senha_1 != Senha_2:   ## validação
            print('As senhas não são iguais')
            return redirect('cadastrar')

        if User.objects.filter(username=Nome_2).exists():   ## validação
            print('Usuário já cadastrado')
            return redirect('cadastrar')
        
        user = User.objects.create_user(username=Nome_2, email='', password=Senha_2) <---- CRIAÇÃO DO OBJETO
        user.save()

        return render(request, 'cadastros.html')
    return render(request, 'cadastros.html')

## Para aquela ultima validação funcionar, temos que importar o modelo de usuarios do Django nessa mesma views.py

	from django.contrib.auth.models import User

Perceba que a identação da variavel "user" é no IF principal e não nos secundários

Pronto, com algumas pequenas validações, o nosso formulário já consegue criar novos usuarios para serem autenticados
