Verifica se o usuário está logado, caso FALSE, redireciona para uma URL padrão, ou configurável

Para configurar a url de redirecionamento podemos:

1 - @Login_required(login_url='/path/')

ou

2 - em Settings.py  > LOGIN_URL = '/path/'


- este caminho /path/ deve ser uma url existente já cadastrada, caso contrário, retornará page not found
- Normalmente usa-se /index/ , que é a página inicial, mas no nosso caso não tinhamos uma url index/ cadastrada
- Portanto usei '/' no lugar de '/path/'
