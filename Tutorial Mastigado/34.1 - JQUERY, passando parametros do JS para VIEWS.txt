O nosso objetivo é: Passar parâmetros atraves da URL para a nossa views usando o metodo GET.

No nosso exemplo, fizemos isso depois que saíamos da tela de print(cancelando), e após isso os javascript recarregava a página, enviando algum parâmetro para dentro da views.
Com isso nós podiamos personalizar este proximo carregamento de página.

- Quando cancelávamos o layout de impressão, tinhamos no javascript:

	window.onafterprint = function(){
        window.location.href += "?postprint=conf";
  	}

	- Desta forma uma variável - postprint - era criada com um valor qualquer "conf", e ficava disponivel no GET.

- Na nossa views:

	def função_qualquer(request,...)
	   if request.method == 'GET' and 'postprint' in request.GET:
		// faríamos qualquer coisa //

- Para pegarmos o valor de uma variavel no GET fazemos o seguinte:

	var = request.GET.get('postprint','')
