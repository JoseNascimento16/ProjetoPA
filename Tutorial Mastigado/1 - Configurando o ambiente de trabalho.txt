- Crie uma pasta para manter seu código
- Acessando essa pasta em um terminal

	C:\caminho_da_pasta
		\cd (nome_da_subpasta)
		\cd ../ (para voltar uma pasta)

-Crie um ambiente virtual com o seguinte comando:

	python3 -m venv ./venv (retira o 3 do python caso dê erro)

- Ative seu ambiente virtual com o seguinte comando:

	venv\Scripts\activate.bat

- Instale o Django nesse ambiente virtualizado:

	pip install django

- Crie um projeto chamado "SetupPrincipal" utilizando o Django admin, para manter toda configuração de nossa   aplicação:

	django-admin startproject SetupPrincipal

- Na pasta setup, altere no arquivo settings.py o idioma e o horário que usaremos na aplicação, incluindo as seguintes linhas de código:

	LANGUAGE_CODE = 'pt-br'          (linha 108)
	TIME_ZONE = 'America/Sao_Paulo'  (linha 110)

	import os (linha 14)

- Botão direito na pasta SetupPrincipal
	Criar nova pasta 'templates'

- Ainda na pasta "setupPrincipal" > settings.py, para finalizar a configuração do ambiente, vamos configurar um diretório para manter os arquivos HTML incluindo a seguinte linha de código em TEMPLATES

	'DIRS': [os.path.join(BASE_DIR, 'SetupPrincipal/templates')],   (linha 59)

- Ao final deste processo, na pasta aplicacao2 teremos:

	- Uma pasta SetupPrincipal
	- Uma pasta VENV
	- Um arquivo manage.py

- Agora já conseguimos rodar o comando:

	python manage.py runserver

- E já estamos prontos para começar!!
